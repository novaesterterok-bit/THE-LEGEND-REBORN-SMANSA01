# KAWANUA360 - Simfoni Digital Sulawesi Utara

Aplikasi Media Pembelajaran dan Informasi Interaktif berbasis Web Responsif yang mengintegrasikan ragam kekayaan pariwisata, kuliner, dan budaya Sulawesi Utara.

## Tim Pengembang: THE LEGEND REBORN SMANSA MANADO
* **Guru Pembimbing:** Nova Ester Terok
* **Ketua Tim:** Pricillia Pontoh
* **Anggota 1:** Zefanya Sumarandak
* **Anggota 2:** Belicia Wawointana

## Fitur Utama:
1. Ensiklopedi Digital (20 Wisata & 20 Kuliner Otentik Sulut).
2. Sistem Pencarian & Filter Cerdas.
3. Gamifikasi Edukasi (Kawanua Image Match & Susun Kata Scramble).
4. Optimalisasi Tampilan Responsif Layar HP.
5. Audio SFX & Musik Latar (BGM) Lagu Daerah.
